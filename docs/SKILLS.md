## 🎨 前端开发

### frontend-specialist

**描述**: 提供前端开发、UI 实现、移动应用开发和现代前端框架能力。

**使用场景**:
- 实现用户界面和交互功能
- 构建可复用的前端组件
- 开发移动端应用（React Native、Flutter）
- 优化前端性能和用户体验
- 处理状态管理和路由

**核心能力**:
- React/Vue/Angular 组件开发
- CSS-in-JS、Tailwind CSS 样式实现
- 前端状态管理（Redux、MobX、Zustand）
- 移动端开发和响应式设计
- 性能优化和代码分割

**示例**:
```bash
# 场景：实现一个用户认证流程的前端界面
使用 frontend-specialist 实现登录表单、验证逻辑和路由守卫
```

## 使用建议

### 选择合适的 Skill

1. **明确任务类型**: 首先识别任务属于哪个技术领域
2. **查看核心能力**: 检查 Skill 的核心能力是否匹配需求
3. **参考示例**: 查看示例场景是否与您的任务相似
4. **组合使用**: 复杂任务可以组合多个 Skills

### 调用方式

```bash
# 方法 1: 在对话中直接请求
"使用 backend-specialist 帮我实现用户认证 API"

# 方法 2: 使用斜杠命令
/backend-specialist

# 方法 3: 在任务描述中提及
"我需要优化数据库查询性能（使用 data-specialist）"
```

### 最佳实践

- **提供上下文**: 描述清楚任务背景和目标
- **明确需求**: 说明期望的输出和约束条件
- **逐步验证**: 对于复杂任务，分步骤验证结果
- **反馈迭代**: 根据输出结果提供反馈，持续改进
